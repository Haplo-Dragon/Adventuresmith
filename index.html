Adventuresmith

This is an Android app for randomly generating elements for tabletop RPGs (names, spells, items, monsters, characters, etc).

I'm not planning to turn this into a general-purpose random table engine. But, I am interested in complicated random generation involving multiple dice, interrelated results from multiple tables, etc.

Bugs? Feature Requests? You can provide feedback here: https://github.com/stevesea/Adventuresmith/issues

Downloads (Beta)

I haven't published the app on Google Play yet. To get a copy of the application, follow this link: https://betas.to/guJ4ZNqt

That link will take you to a sign up page where you enter your email. You'll receive an email with a link to download the Crashlytics Beta app. (you'll need to open the link on your Android phone, using Chrome as your browser)

Once Beta is installed, it'll give you an interface to download Adventuresmith. Beta will download the app and then it should take you to the typical APK install screen where you can choose whether to accept the install and be shown the permissions the app is requesting.

As I fix bugs and implement features, you'll be notified when i publish new beta versions and can choose to install them or not.
